🫀 Heart Disease Prediction



머신러닝 기반 심장질환 예측 프로젝트입니다.  

환자의 건강 데이터를 활용하여 심장질환 여부를 예측하는 분류 모델을 구현했습니다.



📌 Project Goal

\- 심장질환 여부 예측

\- 머신러닝 모델 성능 비교

\- 의료 AI의 조기 진단 보조 가능성 확인



📊 Dataset

\- Heart Disease Cleveland Dataset

\- Target: condition

\- Class 0: 정상

\- Class 1: 심장질환



🤖 Models

\- Logistic Regression

\- SVM

\- Random Forest



📈 Result

Logistic Regression : \*\*0.92\*\* 

SVM : 0.90

Random Forest :  0.88



🏆 Best Model

👉 Logistic Regression



📊 Evaluation

\- Accuracy: \*\*0.92\*\*

\- Precision: \*\*1.00\*\*

\- Recall: \*\*0.82\*\*



📌 Conclusion

Logistic Regression 모델이 가장 높은 정확도를 보였다.  

다만 일부 심장질환 환자를 정상으로 예측하는 경우가 있어, 향후 Recall 개선이 필요하다.
